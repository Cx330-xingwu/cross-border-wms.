# 跨境电商仓库管理系统 (Cross-Border WMS)

专业的跨界电商仓储作业管理平台，支持从包裹入库到报关出库的全流程管理。

## 📁 项目结构

```
cross-border-wms/
├── index.html              # 系统首页（入口）
├── admin/                  # 管理员端
│   ├── index.html         # 管理员页面
│   └── admin.js           # 管理员逻辑
├── warehouse/              # 仓库作业端
│   └── index.html         # 仓库作业页面（含内联JS）
├── shared/                 # 共享资源
│   └── data.js            # 模拟数据 & 工具函数
├── assets/                 # 静态资源
│   ├── css/
│   │   ├── variables.css  # CSS变量（设计令牌）
│   │   └── base.css       # 基础样式 & 组件
│   └── img/               # 图片资源
└── README.md              # 项目说明
```

## 🚀 快速开始

1. 解压项目文件
2. 使用任意 HTTP 服务器打开（或直接双击 `index.html`）
3. 推荐使用 VS Code 的 Live Server 插件

```bash
# 或使用 Python 简易服务器
cd cross-border-wms
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 📋 功能模块

### 管理员端 (`/admin/`)

| 模块 | 功能说明 |
|------|---------|
| 📊 数据看板 | 订单统计、处理趋势、状态分布、最新动态 |
| 📦 订单管理 | 订单列表、搜索筛选、详情查看、状态变更 |
| 🛃 报关管理 | 待报关/已提交/已通过分类管理，一键提交报关 |
| 👥 操作员管理 | 操作员列表、状态监控、今日处理量 |
| 🚚 物流渠道 | 跨境物流合作伙伴管理（DHL/FedEx/UPS等） |
| 📈 数据报表 | 总订单量、申报价值、目的地分布、类别统计 |

### 仓库作业端 (`/warehouse/`)

| 步骤 | 操作说明 |
|------|---------|
| 1️⃣ 扫码入库 | 扫描国内快递单号 / 手动输入查询 |
| 2️⃣ 撕除旧标 | 撕除包裹上的国内快递面单 |
| 3️⃣ 核实商品 | 逐项核对SKU、数量、质量 |
| 4️⃣ 贴跨境面单 | 打印并粘贴跨境物流面单 |
| 5️⃣ 报关提交 | 提交报关资料，等待海关审核 |

## 🎨 技术栈

- **前端：** 纯 HTML5 + CSS3 + Vanilla JavaScript（零依赖）
- **设计：** CSS Variables 设计系统、响应式布局
- **兼容：** 现代浏览器（Chrome/Firefox/Safari/Edge）
- **数据：** LocalStorage 模拟持久化（可对接后端API）

## 📱 响应式支持

- 💻 桌面端（1200px+）
- 📱 平板端（768px - 1024px）
- 📱 移动端（< 768px）

## 🔄 业务流程

```
国内快递到达 → 扫码入库 → 撕除国内面单 → 核实商品
    → 贴跨境物流面单 → 提交报关 → 海关审核 → 出库发货
```

## 📝 备注

- 本系统为前端演示版本，使用模拟数据
- 可通过修改 `shared/data.js` 对接真实后端 API
- 管理员端和仓库作业端可独立部署
- 所有样式使用 CSS 变量，便于主题定制
